Algorithm Visualizer

HTML CSS JavaScript

The Algorithm Visualizer is an interactive web application built to demonstrate the working of sorting and searching algorithms through step-by-step animations. It is designed to make the understanding of algorithm logic and flow easier for learners by providing real-time visual feedback.

Key Features:

Sorting Visualizer:
Supports Insertion Sort, Bubble Sort, Quick Sort, Selection Sort, and Merge Sort.
Animates comparisons and swaps with clear color-coded bars. Demonstrates Linear Search and Binary Search.
Adjustable array size and visualization speed for a customized learning experience.

Searching Visualizer:
Highlights the process of element comparison at each step.
Visually shows how Binary Search divides the array to locate the target.

User Navigation:
Index Page acts as the entry point where users can choose between Sorting Visualizer or Searching Visualizer.
Clean, simple, and beginner-friendly UI.

Technical Highlights:

Frontend: Developed using HTML, CSS, and JavaScript.
Visualization Logic: Implemented with DOM manipulation and timed animations for smooth step-by-step execution.
Learning Utility: Makes abstract concepts like time complexity and search/sort mechanics easier to understand through live demonstrations.

This project is a practical learning tool for students and enthusiasts exploring Data Structures and Algorithms, showcasing skills in JavaScript, problem-solving, and interactive UI development.
