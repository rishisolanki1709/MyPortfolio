Tic Tac Toe Game

HTML CSS JavaScript

The Tic Tac Toe Game is an interactive web-based application that allows two players to play the classic 3Ã—3 board game in a dynamic and engaging way. The project goes beyond the basic version by enabling player swapping and real-time score tracking, making it more competitive and fun.

Key Features:

Two-Player Mode:
Players take turns marking X and O.
Automatic swap of turns between Player 1 and Player 2.

Win Detection & Scoreboard:
Detects winning combinations (rows, columns, diagonals).
Displays the winner instantly after each round.
Maintains a live scoreboard to track the number of wins for both players.

Interactive UI:
Highlighted moves and winning patterns.
Instant reset option to start a new round without refreshing the page.
Responsive and clean design for better gameplay experience.

Technical Highlights:

Frontend: Built using HTML, CSS, and JavaScript.
Game Logic: Implemented with JavaScript functions to handle turn switching, win detection, and score counting.
User Experience: Interactive board with smooth updates to scoreboard and status messages.

This project not only recreates a classic game but also introduces dynamic score tracking and enhanced interactivity, making it a practical demonstration of JavaScript, DOM manipulation, and game development basics.
