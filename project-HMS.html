Velvet Vista Hotel Management Project

Java Spring Boot JPA MySQL Thymeleaf HTML CSS JavaScript

The Hotel Management System is a full-stack web application built using Spring Boot, JPA (Hibernate), REST APIs, and Thymeleaf. It is designed to streamline hotel operations by providing dedicated portals for customers, employees, and administrators. The application follows a layered architecture with controllers, services, repositories, and entity classes, ensuring scalability and maintainability.

Key Features:

Customer Module:
New user registration and login.
Browse available rooms with details.
Online booking and booking history tracking.
Email notifications for successful bookings.

Employee Module:
Manage offline bookings on behalf of customers.
Check room availability and update booking records.
View and manage customer details.

Admin Module:
Add, update, or remove employees and rooms.
Manage admin accounts.
View all booking details and hotel operations at a glance.

Technical Highlights:

Backend: Spring Boot with RESTful controllers and service layers.
Database: MySQL with JPA/Hibernate for ORM and repository-based queries.
Frontend: Thymeleaf templates with HTML, CSS, and JavaScript.
Email Service: Integrated email notifications for booking confirmations.
Architecture: Follows MVC pattern with clear separation of concerns.

This project demonstrates real-world hotel management workflows by integrating booking, staff, and administrative functionalities into a single platform. It emphasizes modular design, database-driven operations, and professional UI integration.
